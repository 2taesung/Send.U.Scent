## Build & Deploy
1. Frontend
    ```sh
    $ cd frontend
    $ npm install
    $ npm run build
    ```
2. Backend
    ```sh~
    $ cd backend
    # application.* 파일들 권한 sudo로 설정되어있기 때문에
    # sudo 권한으로 빌드
    $ sudo mvn package
    $ java -jar target/*.jar --server.servlet.context-path=/api
    ```
